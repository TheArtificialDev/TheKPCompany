# TheKPCompany Website

A modern, comic-style website for TheKPCompany, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Comic-style black and white design with rounded corners
- Subtle animations throughout the site
- Fully responsive layout
- Built with modern web technologies
- Optimized for performance

## Pages

1. **Landing Page**
   - Hero section with title, subtext, and CTA buttons
   - Features section
   - Call-to-action section

2. **Products Page**
   - Mission statement
   - Products grid with search functionality

3. **Why TheKPCompany**
   - Design philosophy
   - Code values
   - Comparative matrix
   - Impact statistics
   - Testimonials

4. **About Us**
   - Origin story
   - Founder bios
   - Company values
   - Company timeline

5. **Careers**
   - Company culture
   - Current openings
   - Benefits & perks
   - Life at TheKPCompany

6. **Blog**
   - Search bar
   - Featured articles
   - Article grid
   - Category filtering

7. **Contact**
   - Contact form
   - Support information
   - Social media links
   - FAQ section

## Technologies Used

- **Next.js**: React framework for server-side rendering and static site generation
- **TypeScript**: For type-safe JavaScript
- **Tailwind CSS**: For styling
- **Framer Motion**: For animations
- **GSAP**: For advanced animations
- **React Icons**: For iconography

## Getting Started

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/thekpcompany.git
   ```

2. Navigate to the project directory:
   ```
   cd thekpcompany
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the site.

## Build for Production

To build the site for production:

```
npm run build
```

To start the production server:

```
npm start
```

## Project Structure

- `src/app`: Next.js App Router pages
- `src/components`: Reusable React components
  - `layout`: Layout components (Navbar, Footer)
  - `ui`: UI components (Button, Card, Section, etc.)
- `src/lib`: Utility functions and hooks
- `src/types`: TypeScript type definitions
- `public`: Static assets
